# 🎴 TCG 卡牌价格查询 Bot

你的智能 TCG 卡牌助手！只需上传卡牌图片，即可获取详细信息和价格查询。

---

## ✨ 主要功能

### 📸 AI 卡牌识别
- **一键识别**：上传卡牌截图，AI 自动识别卡牌信息
- **多游戏支持**：宝可梦 (Pokémon)、海贼王 (One Piece)、游戏王 (Yu-Gi-Oh)
- **智能分析**：基于 Gemini Vision API 的先进图像识别技术

### 🤖 AI 智能分析
- **卡牌效果**：详细解读卡牌技能和效果
- **收藏价值**：⭐ 星级评分 + 价值评估（收藏级珍品/高收藏价值/中等收藏价值等）
- **市场热度**：🔥 人气分析（超热门/热门/一般/冷门）
- **竞技使用**：🏆 环境强度评估（常用/偶尔使用/娱乐卡）
- **亮点特色**：✨ 卡牌卖点总结

### 📊 价格查询
- **智能搜索**：提供 Google 搜索链接快速查询价格
- **多种选择**：仅角色名搜索 / 完整信息搜索
- **市场链接**：TCGPlayer、CardMarket 等主流交易平台

---

## 🚀 如何使用

### 方法一：使用斜杠命令
```
/price [上传卡牌图片]
```

**可选参数**：
- `game`：指定游戏类型（宝可梦/海贼王/游戏王）

**示例**：
```
/price [图片] 海贼王
```

### 方法二：直接发送图片
在任意频道发送卡牌图片，bot 会自动识别并回复！
