# 🎴 TCG 卡牌价格查询 Bot

你的智能 TCG 卡牌助手！只需上传卡牌图片，即可获取详细信息和价格查询。

---

## ✨ 主要功能

### 📸 AI 卡牌识别
- **一键识别**：上传卡牌截图，AI 自动识别卡牌信息
- **多游戏支持**：宝可梦 (Pokémon)、海贼王 (One Piece)
- **智能分析**：基于 Gemini Vision API 的先进图像识别技术

### 🔍 卡牌编号搜索
- **精确搜索**：通过卡牌编号快速查询信息
- **多版本显示**：自动显示同一编号的不同版本（平行、周年纪念等）
- **实时价格**：显示市场价、最低价等价格信息
- **高清图片**：展示卡牌官方图片

### 🤖 AI 智能分析
- **卡牌效果**：详细解读卡牌技能和效果
- **收藏价值**：⭐ 星级评分 + 价值评估（收藏级珍品/高收藏价值/中等收藏价值等）
- **市场热度**：🔥 人气分析（超热门/热门/一般/冷门）
- **竞技使用**：🏆 环境强度评估（常用/偶尔使用/娱乐卡）
- **亮点特色**：✨ 卡牌卖点总结

### 📊 价格查询
- **智能搜索**：提供 Google 搜索链接快速查询价格
- **多种选择**：仅角色名搜索 / 完整信息搜索
- **市场链接**：TCGPlayer、CardMarket 等主流交易平台

## 🚀 如何使用

### 📸 扫描卡牌 - `/Scan`
上传卡牌图片，AI 自动识别卡牌信息、价格和收藏价值。

**斜杠命令**：
```
/Scan [上传卡牌图片]
```

**可选参数**：
- `game`：指定游戏类型（Pokémon / One Piece）

**文本命令**：
```
!scan [回复卡牌图片]
```

**示例**：
```
/Scan [图片] One Piece
```

### 🔍 搜索卡牌 - `/Search`
通过精确卡牌编号快速查询卡牌信息和价格。

**斜杠命令**：
```
/Search [游戏] [卡牌编号]
```

**支持的格式**：
- **One Piece**: `OP01-001`, `OP10-005`, `EB01-001`
- **Pokémon**: `swsh4-136`, `sv1-1`, `base1-4`

**文本命令**：
```
!search OP01-001
```

**示例**：
```
/Search One Piece OP01-001
```

### 📷 自动扫描
在 **Card-Pulls** 频道发照片，bot 会自动跳出来识别！
